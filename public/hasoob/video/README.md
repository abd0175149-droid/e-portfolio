# ضع فيديو التعريف هنا

اسم الملف: `intro.mp4`

## المواصفات الموصى بها:
- الصيغة: MP4
- الدقة: 1280x720 (HD) أو 1920x1080 (Full HD)
- المدة: 2-5 دقائق
- الحجم: أقل من 50 ميجابايت (للتحميل السريع)
- Codec: H.264

## محتوى الفيديو المقترح:
- مقدمة شخصية موجزة
- خلفيتك الأكاديمية والمهنية
- اهتماماتك في مجال تقنيات التعليم
- أهدافك المستقبلية
