# ملف README للمجلد hasoob

## نظرة عامة

هذا المجلد (`hasoob`) يحتوي على جميع الملفات الأصلية لملف الإنجاز الإلكتروني.
جميع الملفات هنا متاحة للتحميل المباشر من خلال واجهة الموقع.

## هيكل المجلد

```
hasoob/

│   └── intro.mp4       # الفيديو التعريفي

├── reflections/        # التفكيرات التأملية
│   ├── reflection_01.docx
│   ├── reflection_02.docx

    └── iste.pdf
```

## ملاحظات مهمة

### 1. أسماء الملفات
- يجب أن تكون أسماء الملفات باللغة الإنجليزية (بدون مسافات)
- استخدم underscore (_) بدلاً من المسافات
- تجنب الأحرف الخاصة والرموز

### 2. صيغ الملفات المدعومة
- **الفيديو**: .mp4, .webm, .avi
- **المستندات**: .docx, .doc, .pdf
- **الصور**: .jpg, .png, .gif (إن وُجدت)

### 3. حجم الملفات
- حاول أن تبقي حجم الملفات معقولاً للتحميل السريع
- الفيديو: يُفضل أقل من 50 MB
- المستندات: يُفضل أقل من 10 MB لكل ملف

## إضافة ملفات جديدة

### الخطوات:

1. حدد المجلد المناسب حسب نوع الملف
2. ضع الملف في المجلد المناسب
3. حدّث ملف JSON المقابل في مجلد `data/` إذا لزم الأمر
4. تأكد من صحة المسار في الكود

### مثال:

لإضافة تفكير تأملي جديد:

1. ضع الملف في `hasoob/reflections/reflection_05.docx`
2. أضف البيانات في `data/reflections.json`:

```json
{
  "id": 5,
  "title": "التفكير التأملي الخامس: ...",
  "date": "2024-11-01",
  "summary": "...",
  "fullContent": "...",
  "file": "reflections/reflection_05.docx"
}
```

## التحقق من الملفات

قبل الرفع إلى Hostinger، تأكد من:

- ✅ جميع الملفات موجودة في المجلدات الصحيحة
- ✅ أسماء الملفات تطابق ما هو مكتوب في ملفات JSON
- ✅ جميع الملفات تفتح بشكل صحيح
- ✅ حجم الملفات مناسب

## الخصوصية والأمان

- جميع الملفات في هذا المجلد ستكون متاحة للتحميل العام
- لا تضع أي ملفات حساسة أو شخصية هنا
- تأكد من حقوق النشر لأي محتوى تضيفه

## استبدال الملفات

إذا كنت تستخدم ملفات نموذجية وتريد استبدالها بملفاتك الحقيقية:

1. احذف الملف النموذجي
2. ضع ملفك الحقيقي بنفس الاسم
3. أو غيّر اسم الملف وحدّث المرجع في الكود
